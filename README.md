What is the use of the keyof keyword in TypeScript? Provide an example.

আমরা অনেক সময় এমন ফাংশন বানাই যা অবজেক্ট-এর নির্দিষ্ট কী নিয়ে কাজ করে, এবং keyof দিয়ে আমরা নিশ্চিত হই যে ভুল কী নেওয়া হবে না।

type Student = {
  name: string;
  age: number;
  isStudent: boolean;
};

type FemaleStudent = keyof Student;


এখানে FemaleStudent এর টাইপ হবে name | age | isStudent
Student টাইপের সব কী FemaleStudent টাইপের ইউনিয়ন হয়।
